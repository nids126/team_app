Team Members Information Portal

This is a portal to view team members information.

URLs for CRUD operation 

1. List all team members

URL : http://127.0.0.1:8000/members

Request Type: GET

2. Add New Team Member

URL : http://127.0.0.1:8000/members/create

Request Type: POST

Request Body: 

{
      "firstName": "First Name" <required>,
      "lastName": "Last Name"<required>,
      "phoneNumber": "Phone Number" <required>,
      "email": "Email Address" <required>,
      "role": "[ADMIN/REGULAR] <optional> {default: REGULAR}"
}

3. View Team Member

URL : http://127.0.0.1:8000/members/<member_id>

Request Type: GET

4. Update Existing Team Member

URL : http://127.0.0.1:8000/members/update/<member_id>

Request Type: POST

Request Body: {
      "firstName": "Updated First Name",
      "lastName": "Updated Last Name",
      "phoneNumber": "Updated Phone Number",
      "email": "Updated Email Address",
      "role": "Updated Role"
}

5. Delete Existing Team Member

URL : http://127.0.0.1:8000/members/delete/<member_id>

Request Type: DELETE
